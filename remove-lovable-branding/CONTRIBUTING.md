# Contributing to Remove Lovable Branding Power

Thanks for your interest in contributing! This power helps developers clean their React apps, and your input makes it better.

## How to Contribute

### Report Issues
Found a Lovable trace this power doesn't catch?
1. Open a GitHub issue
2. Describe where the trace appears (file type, location, pattern)
3. Include an example if possible

### Suggest Improvements
Have ideas for better workflows or search patterns?
1. Open a GitHub issue with the "enhancement" label
2. Describe your suggestion
3. Explain why it would be helpful

### Submit Code
Want to improve the documentation or add new patterns?
1. Fork the repository
2. Make your changes to `POWER.md`
3. Test your changes with a real React app
4. Submit a pull request with a clear description

## Guidelines

- Keep instructions clear and beginner-friendly
- Test all search patterns and commands
- Maintain the step-by-step workflow structure
- Add examples where helpful
- Update the checklist if adding new steps
